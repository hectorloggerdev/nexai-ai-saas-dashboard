# NexAI - AI SaaS Dashboard React Template

NexAI is a premium frontend-only AI SaaS dashboard template built with React, Vite, TypeScript, TailwindCSS, Recharts, Lucide icons, and mock data.

## Features

- AI SaaS landing page
- Login and register screens
- Responsive dashboard layout
- Mobile sidebar navigation
- Dark and light mode
- Theme persistence with localStorage
- Dashboard analytics
- AI Chat page
- AI Image Generator page
- Vision Analysis page
- Usage and limits page
- Users page with reusable data table
- Billing and pricing page
- Invoice history page
- Notifications page
- API Keys page
- Settings page
- Profile page
- 404 page
- Mock data only; no backend required

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

This is a frontend template. Backend, authentication service, payment provider, database, and AI provider API integrations are not included.
