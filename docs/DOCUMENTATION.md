# NexAI Documentation

## Overview

NexAI is a frontend-only React template for AI SaaS dashboard products.

## Pages

Landing, Login, Register, Dashboard, AI Chat, Image Generator, Vision Analysis, Usage & Limits, Users, Billing, Invoices, Notifications, API Keys, Settings, Profile and 404.

## Customization

Colors are configured in `tailwind.config.js`. Mock records are located in `src/data/mockData.ts`. Theme logic is located in `src/context/ThemeContext.tsx`.

## Backend

No backend is included. Buyers can connect their own backend/API.
