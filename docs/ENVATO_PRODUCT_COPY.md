# Envato Product Copy
## Title
NexAI - AI SaaS Dashboard React Template
## Short Description
Premium React AI SaaS dashboard template with AI chat, image generator, vision analysis, users, billing, invoices, API keys, integrations, usage analytics, responsive layout, and dark/light mode.
## Key Features
- React + Vite + TypeScript
- TailwindCSS design system
- Premium SaaS landing page
- Responsive dashboard layout
- Dark and light mode
- Theme persistence
- AI Chat page
- Image Generator page
- Vision Analysis page
- Usage limits dashboard
- Users table
- Invoices table
- Notifications UI
- Integrations UI
- API Keys page
- Settings and Profile pages
- 404 page
- Recharts analytics
- Framer Motion hero animation
- No backend required
